# Arduino-Sequencer
16 step digital sequencer made for an Arduino Nano | 
Video overview - https://www.youtube.com/watch?v=8aXAuAVppR0

Uses an Arduino Nano, a rotary encoder button, potentiometer, 8 blue 5mm LEDs and an LCD screen to create a sequencing interface.
Uses the Arduino's built-in "tone" function to produce sound.
